# Ethereum eth/v1/node/peer_count RPC method

Ethereum consensus layer Beacon Chain API call that returns the number of the peers connected to the node.

**Parameters:** 

* `none`

**Returns:** 

* `data` - `object`:
  * `disconnected`
  * `connecting`
  * `connected`
  * `disconnecting`

**Example:**
