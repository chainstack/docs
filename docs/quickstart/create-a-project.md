# Create a project

You must create a [project](/glossary/project) to be able to deploy a node or a network, or to join a network.

To create a project:

1. Log in to your <a href="https://console.chainstack.com/" target="_blank">Chainstack account</a>.
1. Click **Create project**.
1. Click **Public chain** for an [Ethereum](/blockchains/ethereum), [Bitcoin](/blockchains/bitcoin), or [Corda](/blockchains/corda) node deployment, or **Consortium** for [Hyperledger Fabric](/blockchains/fabric), [Corda](/blockchains/corda), [Quorum](/blockchains/quorum), or [MultiChain](/blockchains/multichain).
1. Provide **Project name** and optionally **Description**.
1. Click **Create**.

This will create a project.
